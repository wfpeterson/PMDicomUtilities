Configure the system by opening Edit->DICOM configuration.
Add at least one MWL SCP entries and one Store SCP entries. This will allow you to perform modality wordlist queries and send images to a storage SCP. The test button in the DICOM AE entry dialog allows you to send a DICOM Echo to the AE as configured in the dialog. Multiple destinations can be defined an used.
To simulate a modality interaction first query the MWL SCP by entering search keys and issuing a query, pressing the Query button. If you wish to enter new parameter, the clear button clears all the foods before entry. If you are satisfied without the demographic info, you can select a directory of .dcm files to send. Once the directory is identified the send button allow you to initiate the transmission.
The application is single threaded, You cannot interact with the GUI till the last request is complete.
The File->ImageInfo Menu allows you to inspect a DICOM file. File->Exit closes the application
